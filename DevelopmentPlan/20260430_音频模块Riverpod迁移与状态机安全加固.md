## 三、详细执行记录

### 3.1 Provider 生命周期托管

**问题**：`ttsEngineProvider` 创建 `TtsEngineService` 后未注册 Riverpod 销毁钩子，依赖 `_BootstrapperState.dispose()` 手动调用，存在双调用风险。

**方案**：

```dart
final ttsEngineProvider = ChangeNotifierProvider<TtsEngineService>((ref) {
  final settings = ref.read(settingsProvider);
  final svc = TtsEngineService(settings, externalSettingsListener: false);
  ref.onDispose(svc.dispose);   // ← Riverpod 自动销毁
  ref.listen<SettingsProvider>(
    settingsProvider,
    (_, next) => svc.syncSettingsFromProvider(next),  // ← 声明式配置同步
  );
  return svc;
});
```

**幂等守卫**（防双调用）：

```dart
@override
void dispose() {
  if (_disposed) return;  // ← 新增守卫
  _disposed = true;
  ...
}
```

### 3.2 依赖解耦：ref.listen 替代 addListener

**问题**：`TtsEngineService` 构造函数中 `_settings.addListener(_onSettingsChanged)` 与 Riverpod 的 `ref.listen` 并存，存在双重注册风险。

**方案**：新增 `externalSettingsListener` 参数控制监听策略：

- `false`（Riverpod 场景）：引擎不自行 addListener，由 `ref.listen` 统一推送
- `true`（默认/测试场景）：引擎自行注册内部监听器，维持向后兼容

新增 `syncSettingsFromProvider(SettingsProvider)` 公开方法供 `ref.listen` 调用。

### 3.3 状态机安全加固

**问题**：`TtsPlaybackState` 枚举只有 4 个值，缺少显式错误状态，所有 switch 表达式不完整。

**方案**：新增 `error` 状态分支：

```dart
enum TtsPlaybackState { disabled, paused, buffering, playing, error }
```

新增 `isError` getter：

```dart
bool get isError => _state == TtsPlaybackState.error;
```

**穷尽 switch 升级**（3 处）：

1. **播放循环 finally**：`(isEnabled, isPaused, isError, _currentItem != null)` 4 元组
2. **refreshSession()**：`(isEnabled, isPaused, isError)` 3 元组  
3. **forceRestartLoops()**：`(isEnabled, isPaused, isError)` 3 元组

### 3.4 ReaderProvider 穷尽 switch

**问题**：`toggleTTS()` 使用 if-else 链，末尾兜底 `return TtsToggleResult.playing` 掩盖状态机漏洞，`error` 状态无法响应。

**方案**：重写为 Dart 3 穷尽 switch 表达式，编译期静态验证：

```dart
return switch (_ttsEngine.state) {
  TtsPlaybackState.playing || TtsPlaybackState.buffering => () { ... }(),
  TtsPlaybackState.paused  || TtsPlaybackState.disabled => () { ... }(),
  TtsPlaybackState.error => () {
    _ttsEngine.clearLastError();   // 清除错误后尝试恢复
    _ttsEngine.play();
    return TtsToggleResult.playing;
  }(),
};
```

**readerProvider 生命周期注册**：

```dart
final readerProvider = ChangeNotifierProvider<ReaderProvider>((ref) {
  final rp = ReaderProvider(ref.read(ttsEngineProvider));
  ref.onDispose(rp.dispose);  // ← 新增
  return rp;
});
```
