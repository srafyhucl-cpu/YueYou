# 阅游 (YueYou) 20260430 任务汇总：Riverpod 全域迁移与规范落地

## 1. Riverpod 架构迁移 (V1.1 Phase 1 & 2)

### 任务目标
完成从 `provider` 到 `flutter_riverpod` 的全面架构升级，彻底移除旧版依赖。

### 完成情况
- **基础设施**：引入 `flutter_riverpod: ^2.6.1`，根部包裹 `ProviderScope`。
- **Provider 迁移**：
  - `SettingsProvider` / `BookshelfProvider` (Phase 1)
  - `TtsEngineService` / `GameProvider` / `ReaderProvider` (Phase 2)
- **UI 迁移**：全面重构 `library_screen`、`settings_screen`、`SquareBoard`、`BoardMascot`、`TeleprompterView` 等组件为 `ConsumerWidget` 或 `ConsumerStatefulWidget`。
- **清理与适配**：彻底移除 `provider` 包，更新所有 Widget 测试至 `ProviderScope` 覆盖注入，解决跨文件并发竞态导致的测试不稳定性。

### 架构终态
```
ProviderScope
├── settingsProvider      (ChangeNotifierProvider)
├── bookshelfProvider     (ChangeNotifierProvider)
├── ttsEngineProvider     (ChangeNotifierProvider)
├── gameProvider          (ChangeNotifierProvider)
└── readerProvider        (ChangeNotifierProvider)
```
