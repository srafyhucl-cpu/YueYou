# 全局性能优化 P0 修复

## 背景

经过多轮全局性能深度审计（presentation/providers/services/domain/core/CustomPainter/AnimationController/Shader），
从全局用户感知视角重新分级，最终确认 4 项 P0 关键性能问题，本次一次性集中修复。

## 任务清单

### P0-A + P0-B：灵动岛（CyberPlayerConsole + NeonProgressPainter）

- [x] `dashboard_screen.dart`：灵动岛加 `RepaintBoundary` 隔离呼吸动画脏区
- [x] `cyber_player_console.dart`：呼吸动画按 TTS 状态（playing/buffering）启停，idle 时停止 60fps 驱动
- [x] `neon_progress_painter.dart`：4 个 Paint 实例静态复用 + LinearGradient shader 按 (size, color) 缓存

### P0-C：_MascotFacePainter RadialGradient shader 缓存

- [x] `board_mascot.dart`：`_drawCore` 中 RadialGradient shader 按 hasError 缓存，避免每帧 GPU shader 创建

### P0-D：CurvedAnimation 泄漏修复

- [x] `square_board.dart`：`_triggerTilt` 复用单个 `CurvedAnimation` + 目标值字段，不再每次滑动新建
- [x] `board_mascot.dart`：5 处 CurvedAnimation 持久复用（eye/body/expression/wobble/pulse），
  `_moveEyes`/`_triggerWobble`/`_setExpression` 仅更换 Tween 包装（轻量 `_AnimatedEvaluation`），dispose 统一清理

## 验收标准

- [x] `flutter analyze` 零错误零警告
- [ ] dashboard 静置 1 分钟 GPU 平均 < 1ms/帧
- [ ] 游戏连续滑动 30 次 GPU 峰值 < 12ms/帧
- [ ] 30 分钟游戏后 AnimationController listener 数量稳定
- [ ] 所有现有视觉效果像素级一致
