# 技能名称：纯净领域逻辑与矩阵算法专家

## 核心职责

你现在进入了“无 UI 模式”。你的任务是编写 `features/*/domain/` 目录下的纯核心业务逻辑（例如 2048 的 4x4 矩阵合并算法）。

## 强制执行规范

1. **UI 物理隔离**：在这个技能卡激活期间，生成的代码中**严禁引入 `package:flutter/material.dart`**。你只能使用 Dart 的核心库（如 `dart:math`、`dart:collection`）。
2. **纯函数设计**：所有的矩阵滑动、合并算法，必须是输入一个状态、返回一个新状态的“纯函数 (Pure Function)”，严禁在算法内部直接修改外部全局变量，必须保证代码是 100% 可单元测试的。
3. **强类型数据结构**：二维数组必须明确声明泛型（如 `List<List<int>>`），严禁使用 `dynamic`。
4. **中文逻辑注释**：在复杂的算法步骤前（例如：1. 去除空格，2. 合并相同数字，3. 补齐空位），必须用中文清晰地写出推演步骤注释。
