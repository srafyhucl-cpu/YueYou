## 本版重点

- 清理历史泄密风险：重写 Git 历史，移除旧 Android 签名密码和本机 JKS 路径。
- 启动隐私闸门：未同意隐私协议前不初始化完整业务栈。
- TTS 安全治理：服务端不记录朗读原文，TTS 临时音频走私有对象和短效签名 URL。
- Android 本地数据边界：禁用系统自动备份和 Android 12+ 数据提取。
- 测试与 CI 门禁：Flutter 分析、测试目录分析、AI 工程门禁、覆盖率、Go 服务端测试和 race 检查进入 CI。
- 发布治理：新增手动 arm64 Release APK 构建任务，版本、Sentry release 和 APK versionName/versionCode 同源。

## 回滚说明

如 `v1.1.1` 发布后发现阻断问题：

1. 在应用市场停止继续分发 `v1.1.1` APK。
2. 保留 `v1.1.1` Tag 与 artifact，不删除发布证据。
3. 从 `main` 建立修复分支，完成修复后递增到 `1.1.2+4`，禁止覆盖 `v1.1.1`。
4. 若必须撤回服务端能力，优先回滚 `/www/wwwroot/yueyou/yueyou-server` 二进制并重启 `yueyou.service`。
5. 如 TTS 对象访问或签名 URL 策略异常，先关闭或收紧服务端 TTS 入口，再补发客户端修复版。
