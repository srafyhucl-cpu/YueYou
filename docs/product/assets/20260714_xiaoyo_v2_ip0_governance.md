## 当前结论

当前仅有方向候选图和 Rive Community 兼容资源，IP-0 状态为 `candidate`，商业使用
批准为 `false`。候选图用于讨论角色方向，社区 `assets/rive/xiaoyo.riv` 只用于既有
输入兼容核对；两者都不能作为原创商业母版、版权登记材料或对外宣传主资产。

机器可读状态位于
`docs/product/assets/20260714_xiaoyo_v2_ip0_manifest.json`，可用以下命令检查：

```powershell
dart run scripts/check_xiaoyo_ip0.dart
```

命令默认只检查清单内部是否自相矛盾。后续 Rive 接入前必须使用严格门禁：

```powershell
dart run scripts/check_xiaoyo_ip0.dart --require-accepted
```

`commercialUseApproved` 改为 `true`。

## 资产目录边界

- 商业母版固定使用独立文件名 `xiaoyo_v2_base.riv`，不得覆盖或改造社区
  `assets/rive/xiaoyo.riv`。
- 每个后续皮肤记录版本、来源、作者、授权范围、依赖的输入契约和静态降级图。
- 不把书名、正文、用户进度、设备标识或支付信息写入资产证据。
- 在 IP-0 通过前，不制作批量皮肤、GLB、Unity 场景或付费商品图。
- 现有版权材料不在本切片范围内，相关口径在原创资产最终完成后单独复核。
