## 2. 状态机流转

```text
                 ┌──────────────────────────┐
                 │       TtsAudioIdle        │
                 │  (空闲 / 引擎未启用)       │
                 └───────────┬──────────────┘
                             │ play() 首次启动
                             ▼
                 ┌──────────────────────────┐
                 │     TtsAudioBuffering     │ ◄──────────────┐
                 │  (缓冲中 / 预加载音频)     │                │
                 └───────────┬──────────────┘                │
                             │ _buffer 有内容                 │
                             ▼                               │
                 ┌──────────────────────────┐                │
                 │      TtsAudioPlaying      │── playFile() ──┤
                 │  (播放中)                  │   完成后继续    │
                 │                           │   消费下一句   │
                 └──────┬───────────────┬────┘                │
                        │               │                    │
                   pause()          stopAll()                 │
                        ▼               ▼                    │
                 ┌───────────┐   ┌───────────┐               │
                 │ Paused    │   │  Idle     │               │
                 │ 暂停，保留 │   │ 完全停止  │               │
                 │ 缓冲 + 句 │   │ 清空缓冲  │               │
                 └─────┬─────┘   └───────────┘               │
                       │ play() 恢复                          │
                       └─────────────────────────────────────┘
                 
                 ┌──────────────────────────┐
                 │       TtsAudioError       │── recover() ──▶ Buffering
                 │  (网络异常 / 播放失败)     │
                 └──────────────────────────┘
```

## 3. 双轨并行的时序

```text
时间线 ──────────────────────────────────────────────────────►

预加载轨道:  下载句1  下载句2  下载句3  ...  下载句4  下载句5  ...
             │        │        │              │        │
播放轨道:           播放句1   播放句2   播放句3        播放句4
                    │        │        │              │
阅后即焚:                   删句1    删句2          删句3

缓冲队列:  [句2句3]  [句3句4]  [句4句5]  [句5句6]  ...
```

## 4. 降级路径

```text
downloadAudio() 连续失败 5 次
       │
       ▼
  _degradeToLocal()
       │
       ├── 1. stopAudio()     ← 停远程播放器，杜绝二重唱
       ├── 2. pauseAudio()    ← 停本地播放器
       ├── 3. _isDegraded = true
       └── 4. speakWithLocalTts()  ← FlutterTts 朗读
              │
              恢复条件: nextTtsSentence() 返回 null →
              _isDegraded = false → 切回远程下载
```
